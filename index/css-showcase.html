CSS Borders


The border-style Property

This property specifies what kind of border to display:

A dotted border.

A dashed border.

A solid border.

A double border.

A groove border.

A ridge border.

An inset border.

An outset border.

No border.

A hidden border.

A mixed border.


The border-width Property

This property specifies the thickness of the border to display:

A 1px border

A 2px border

A 3px border

A 4px border

A 5px border

A 8px border

A 10px border

A 15px border

A 20px border

A border with different widths


The border-color Property

This property specifies the color of the border to display

A red border

A orange border

A yellow border

A green border

A blue border

A purple border

A violet border

A pink border

A gray border

A black border

A multicolored border


The border-left/top/right/bottom Property

This property specifies which side of the border to display


 A border on the left


 A border on the top


 A border on the right


 A border on the bottom



The border-radius Property

This property specifies how round the border will be to display

Normal border

Barely round border

Very Slightly round border

Round border

Rounder border

Very round border

Roundest border


The cursor property

This property specifies what cursor your mouse will be to display


Mouse over the words to change the cursor.


CSS transitions


The transition property

This property makes a transition, Click and hold the shapes to see the transition effect


The transition-timing-function property

This property defines the timing of the transition


timing-linear
timing-ease
timing-ease-in
timing-ease-out
timing-ease-in-out
Custom timing (cubic-bezier(0.1, 0.7, 1.0, 0.1);)

The filter property

This property changes the image's look

Note: The filter property is not supported in Edge 12 or Internet Explorer and the picture may break in other browsers.
